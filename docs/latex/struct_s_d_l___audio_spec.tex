\doxysubsection{Detailed Description}
The calculated values in this structure are calculated by \doxylink{_s_d_l__audio_8h_a2edf30e7747584e28041b4986f89f440}{SDL\+\_\+\+Open\+Audio()}.

For multi-\/channel audio, the default SDL channel mapping is\+: 2\+: FL FR (stereo) 3\+: FL FR LFE (\doxylink{class_vector__2d_a6d09fb691e01f179efcbef03bef13b9c}{2.\+1} surround) 4\+: FL FR BL BR (quad) 5\+: FL FR LFE BL BR (\doxylink{class_vector__2d_a6d09fb691e01f179efcbef03bef13b9c}{4.\+1} surround) 6\+: FL FR FC LFE SL SR (\doxylink{class_vector__2d_a6d09fb691e01f179efcbef03bef13b9c}{5.\+1} surround -\/ last two can also be BL BR) 7\+: FL FR FC LFE BC SL SR (\doxylink{class_vector__2d_a6d09fb691e01f179efcbef03bef13b9c}{6.\+1} surround) 8\+: FL FR FC LFE BL BR SL SR (\doxylink{class_vector__2d_a6d09fb691e01f179efcbef03bef13b9c}{7.\+1} surround)
