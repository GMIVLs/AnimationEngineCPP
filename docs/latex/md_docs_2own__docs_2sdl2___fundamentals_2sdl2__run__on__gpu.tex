\begin{DoxyItemize}
\item The M1 is Apple\textquotesingle{}s ARM-\/based chip that has integrated GPU cores. It\textquotesingle{}s quite powerful and can definitely handle intensive 2D graphics.
\item If you\textquotesingle{}re developing on an M1 Mac, ensure you have the latest SDL2 libraries that are compatible with the ARM architecture of the M1.
\item SDL2 has support for Metal, which is Apple\textquotesingle{}s graphics and compute API. This means SDL2 can take advantage of the M1\textquotesingle{}s GPU capabilities via Metal for rendering.
\end{DoxyItemize}

The SDL2 offers hardware-\/accelerated rendering for 2D graphics, if you need more control over the GPU, especially for 3D graphics or custom shaders, you might want to consider combining SDL2 with a graphics API like Open\+GL, Vulkan, or Metal. SDL2 can be used to create a window and handle input, while the graphics API manages the rendering.
